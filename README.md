# Python Assignment – Part 2 (Order & Inventory System)

## About this project

This is a simple Python program that simulates a basic restaurant system.
It includes ordering, cart handling, inventory tracking, and sales analysis.

I created this project to practice Python concepts like loops, functions, dictionaries, and basic logic building.

---

## Features

1. **Order System**

* User can enter item names to place an order
* Checks if item exists, is available, and in stock
* Calculates total bill

2. **Cart System**

* Add items with quantity
* Remove items from cart
* Display current cart
* Calculates subtotal, tax, and final amount

3. **Inventory Backup**

* Uses deepcopy to create a backup of stock data
* Restores original data after modification

4. **Sales Analysis**

* Calculates total revenue per day
* Finds best selling day
* Finds most ordered item

---

## How to run the program

1. Open the folder in VS Code
2. Open terminal
3. Run the file:

```bash
python part2_order_system.py
```

4. Enter item names when prompted
5. Type `done` to finish ordering

---

## Notes

* Item names must match the menu exactly
* Some items may be unavailable or out of stock
* Cart system is demonstrated using sample function calls
